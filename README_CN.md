# GSC Saver：Google Search Console 数据永久备份方案

Google Search Console (GSC) 仅保留最近 **16 个月** 的搜索数据。如果你不及时归档，这些珍贵的历史 SEO 洞察将会永久丢失。

本仓库提供了两种解决方案：

1. **免费脚本：** 适合希望手动处理 GSC API 的开发者。
2. **GSC Saver 插件：** 一款“买断制”的专业工具，可实现自动化、隐私优先的 Google Drive 数据备份。

------

## 🛑 问题所在：“16个月的数据断崖”

Google 会自动删除超过 16 个月的关键词、点击量和曝光量数据。

- **无法进行同比分析：** 你无法将今年的表现与 2 年前的数据进行对比。
- **高昂的方案：** 企业级 SEO 工具集仅仅为了“归档”你的数据，每月就会收取 **$100+** 的费用。
- **数据所有权：** 你的 SEO 数据不应该有过期时间。

------

## 🛠 免费方案：手动备份脚本

如果你熟悉脚本编写和 API 配置，可以使用本仓库中包含的旧版脚本。

- **功能：** 连接 GSC API，获取数据并保存至 Google 表格 (Google Sheets)。
- **操作文档：** [查看脚本手册 (中文)](https://github.com/MiceXH/GSC-Saver-Doc/blob/main/GSC数据备份操作手册.md)
- **局限性：** 需要一定的技术背景进行设置，需手动执行，且缺乏与 Google Drive 的自动化同步功能。

------

## 🚀 专业方案：GSC Saver 插件

**GSC Saver** 是一款 Chrome 浏览器插件，专为希望拥有“一劳永逸”方案的站长和 SEO 专业人士打造。

### 核心功能：

- **零配置：** 无需 API 密钥，无需编写代码。
- **自动同步：** 直接在 GSC 数据与 **你自己的 Google Drive** 之间建立桥梁。
- **数据所有权：** 无第三方服务器。你的敏感 SEO 数据仅存在于 GSC 和你的 Google Drive 之间。
- **买断制：** 告别订阅焦虑。一次付费，终身保障。

### [👉 在 Chrome 网上应用店获取 GSC Saver](https://chromewebstore.google.com/detail/jafnmmnidbecklolnjlkakhppjmoolfn?utm_source=item-share-cb)

## 🔒 隐私与安全

我们坚信用户应该拥有自己的数据。

- **本地处理：** 插件在你的浏览器本地处理数据。
- **直接对接：** 数据直接从 Google Search Console API 传输到你的 Google Drive。
- **零知识证明：** 我们永远无法看到你的关键词、点击量或站点 URL。

------

## 📖 如何使用（插件版）

1. 从 Chrome 网上应用店安装 **GSC Saver**。
2. 点击 GSC Saver 图标并授权访问你的 Google Drive。
3. 开启“自动同步” (Auto Sync)。
4. 坐享其成，看着你的 SEO 资产在 Google Drive 中持续增长。
